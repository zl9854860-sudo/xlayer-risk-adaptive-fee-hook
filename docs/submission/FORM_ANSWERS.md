## Project Name

Risk Adaptive Fee Hook

## One-Line Description & Project Highlights

Risk Adaptive Fee Hook is a Uniswap v4 dynamic-fee Hook deployed on X Layer mainnet. It lets meme and newly launched asset pools raise swap fees automatically when pool risk increases, giving LPs more compensation during volatile or suspicious trading periods. The project includes a deployed Hook, initialized v4 dynamic-fee pool, risk update transaction, and a real swap transaction where PoolManager emitted fee `6500`, matching the Hook's quoted risk-adjusted fee.

## Any Other Words To X Layer Team

This project explores a practical Hook primitive for high-volatility launch markets on X Layer: risk-aware dynamic fees. The demo is intentionally small and verifiable. It shows the full on-chain path: Hook deployment, v4 dynamic-fee pool initialization, risk update, liquidity provision, and a real swap where the emitted PoolManager fee equals the Hook's quoted risk-adjusted fee. The next product step would connect the risk score to an off-chain agent using market volatility, social spikes, and pool behavior signals.
